
## Features

| Tool | Description |
|------|-------------|
| `vara_scaffold_program` | Scaffold a new Vara/Sails program from templates |
| `vara_compile` | Compile your program to WASM |
| `vara_test` | Run your program's test suite |
| `vara_client_scaffold` | Generate TypeScript client code |
| `vara_docs_search` | Search Vara documentation |

## Getting Started

After installing, open Cursor and use the **"Create a Vara MVP (Counter)"** prompt to:

1. ✅ Scaffold a new Sails counter program
2. ✅ Compile it to WASM
3. ✅ Run tests
4. ✅ Generate a TypeScript client

Just tell Cursor: *"Create a new Vara counter program called my-counter"*

## Requirements

- **Node.js** 18+
- **Rust** with `wasm32v1-none` target
- **Cursor IDE**

```bash
# Install Rust WASM target if needed
rustup target add wasm32v1-none
```

## Environment Variables (Optional)

| Variable | Description |
|----------|-------------|
| `VARA_RPC_URL` | Custom Vara network RPC endpoint |
| `VARA_WORKSPACE_ROOT` | Override default workspace root |
