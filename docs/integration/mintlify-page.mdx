# Vara MCP Server for Cursor IDE

Build Vara Network smart programs faster with AI-assisted development directly in Cursor IDE.

## Manual Installation

<Steps>
  <Step title="Install the package">
    ```bash
    npm install -g vara-mcp-server
    ```
  </Step>
  <Step title="Configure Cursor">
    Create `.cursor/mcp.json` in your project:
    ```json
    {
      "mcpServers": {
        "vara-mcp": {
          "command": "npx",
          "args": ["vara-mcp-server"]
        }
      }
    }
    ```
  </Step>
  <Step title="Restart Cursor">
    Restart Cursor IDE to load the MCP server
  </Step>
</Steps>

## Quick Start

After installing, open Cursor and use the built-in prompts:

<AccordionGroup>
  <Accordion title="Create a Vara MVP (Counter)">
    Tell Cursor: *"Create a new Vara counter program"*
    
    This will automatically:
    1. Scaffold a new Sails counter program
    2. Compile it to WASM
    3. Run tests
    4. Generate a TypeScript client
  </Accordion>
  <Accordion title="Add Feature to Program">
    Tell Cursor: *"Add a multiply feature to my counter"*
    
    The AI will analyze your existing code and suggest patches.
  </Accordion>
</AccordionGroup>

## Requirements

<Info>
  Make sure you have the following installed before using the MCP server:
</Info>

| Requirement | Version | Installation |
|-------------|---------|--------------|
| Node.js | 18+ | [nodejs.org](https://nodejs.org) |
| Rust | Latest stable | [rustup.rs](https://rustup.rs) |
| Build output | - | Scaffold uses `target/wasm32-gear/release/` only (no wasm32-unknown-unknown) |

## Configuration Options

<ParamField path="VARA_RPC_URL" type="string">
  Custom Vara network RPC endpoint for deployment operations
</ParamField>

<ParamField path="VARA_WORKSPACE_ROOT" type="string">
  Override the default workspace root directory
</ParamField>

### Example with Environment Variables

```json
{
  "mcpServers": {
    "vara-mcp": {
      "command": "npx",
      "args": ["vara-mcp-server"],
      "env": {
        "VARA_RPC_URL": "wss://testnet.vara.network"
      }
    }
  }
}
```

## Troubleshooting

<Accordion title="Rust/Cargo not found">
  Install Rust via rustup:
  ```bash
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  source $HOME/.cargo/env
  ```
</Accordion>

<Accordion title="Build output location">
  Scaffolded projects output to `target/wasm32-gear/release/` (`.wasm`, `.opt.wasm`, `.idl`). Run `cargo build --release`.
</Accordion>

<Accordion title="MCP server not loading">
  1. Verify the config file location: `.cursor/mcp.json`
  2. Check JSON syntax is valid
  3. Restart Cursor IDE completely
</Accordion>
